# 30 Days Problem Solving Challenge: Python + AI/ML

## Overview
This repository contains my **30-Day Problem Solving Challenge**:
- **1 Python coding problem**
- **1 AI/ML problem-solving task**

The goal of this challenge is to strengthen:
- Python programming skills
- Logical thinking
- Machine learning problem-solving ability
- Interview communication confidence

---

## Challenge Structure

Each day includes:
### Python Section:
- Problem statement
- Solution code
- Explanation

### AI/ML Section:
- Real-world ML problem/case study
- Approach explanation
- Solution implementation
